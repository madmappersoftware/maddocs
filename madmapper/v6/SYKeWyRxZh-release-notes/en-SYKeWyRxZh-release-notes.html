6.0 Beta 1 Release Notes

ADDITIONS

+ Added Timelines:
    + Timelines are replacing Cues, they are like Cues but all parameters can be animated
    + Visuals can be added in Montage Tracks and used as visual for video / laser layers
    + Audio Track
    + OSC / MIDI Output tracks
    + Markers allow interactivity on timeline playback
    + Conductor is the master timeline, it can be synced to an external Time Code (MIDI MTC / Audio LTC / ArtNet TimeCode)
    + Montage tracks control media playback (Movies, Materials, Generators etc.)
    + Surface FX Timebase can be controlled by timelines playyback
    + Export your timeline to get movies for each Video Output, ILDA file for each Laser Output, Audio File for all audio in timeline (audio tracks & montage tracks), MadLight Sequence for DMX Output
    + Timeline in grid (not conductor) can be synched to BPM with Quantized Launch & BPM Sync Playback. Sync your Video, your LEDs, Lights & Lasers with Ableton Link
+ MadMapper Projects are now stored in a folder with resources (workspace)
    + Menu File / Collect and Save allows you to import all necessary resources (Materials, movies, audio files etc.) so your project can be easiely moved to another computer
+ MadLight: DMX Fixtures are now in the output and sampling video surfaces
+ Support more display scale ratio on Windows (like scaling 150%, before MadMapper supported 100% or 200% but not inbetween)
+ MIDI / OSC outputs can be defined in Project Settings
+ Object Tracks can be pinned (pinned tracks always keep visible when searching text) + filter to show only pinned tracks
+ Support render passes in ISFs
+ Library resources (Materials etc.) are available offline (when no internet)

 

IMPROVEMENTS

* Media Tab is on left side by default, can be extracted on the right side with menu View / Extract Media Tab or with toolbar button
* Fading colors is made in perceptive space (in timeline track/key transitions)
