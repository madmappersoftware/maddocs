2023

MadMapper 5.5

 

MadMapper version 5.5 unveils the MAD Laser Scanner, which turns your camera and laser into a 2D spatial scanning tool!

The update also introduces 4 new modules, the support of HAP-R codec on Windows, a project-backup system, and more.

 

Features

  • Laser scanner
  • AutoSave & Project Backup
  • 4 New Modules for automatically executing a Cue right after opening a project
  • 3D Particles generator
  • HapR Codec support
  • "Adjust movie framerate" option in the MiniMad video's advanced export setting
  • New import of SVG & Export SVG button on masks and lines surfaces
  • Fullscreen button on the main window

2022

MadProjectorControl

 

The extension (which starts at 5.9€/month excl tax) allows you to control your video-projectors remotely through the network using the PJLink protocol (h[url]ttps://pjlink.jbmia.or.jp/english[/url])
With the help of MadMapper’s Calendar Scheduler & Cue Scheduler, you can also fully automate your installation, removing the need for a dedicated operator!

2021

MadMapper 5

 

MadMapper 5 is released with a major new feature: MadLaser, a module for controlling laser beams through the MadMapper interface, using the integrated generative texture library. The extension also enables real-time vectorization of video content.

 

Features

  • MadLaser laser control interface
  • Masks, lines, and surfaces can be shaped by laser
  • Multi protocol laser: Ilda, FB4, Lasercube
  • Real-time video vectorization
  • Laser Dispatch: precise combinations of multiple lasers.
  • Real-time laser texture programming
  • GPU particle module

2020

MadMapper 4

 

MadMapper becomes a Live Coding tool! Version 4.0 implements a range of surface effects as well as a code editor, enabling graphic shaders to be created directly in MadMapper in real-time.

The update features a more comprehensive interface, making the software more accessible and versatile. Masks can now be created using bezier lines and curves, and the texture library is now a shared online library, enabling all users to share and download an exponential number of animated textures.

 

Features

  • New texture library
  • New DMX monitor
  • Multiple masks
  • Surface effects & feathering
  • Surface code editor
  • Connection to Arduino sensors

2017

MadMapper 3

 

The new MadMapper updates are impressive: first and foremost, the major evolution is the introduction of a generative texture library and an editor for coding them.

But this version also marks MadMapper's entry into Windows compatibility, whereas it was developed for MacOS only in its early days, as well as compatibility with Ableton Live software, asserting the link between digital art and musical composition.

Finally, the software adopts a new interface based on the Scene model, facilitating navigation between the various parameters of each project.

 

Features

  • Texture library
  • Texture code editor
  • Ableton link
  • Parametric scene system
  • 3D lighting
  • Live video input
  • Space Scanner: recognizes a 3D volume
  • 3D calibration
  • Video effects
  • Scene settings interface
  • Texture code editor
  • Texture library

2015

MiniMad

 

MiniMad is a stand-alone box that runs a MadMapper session when connected to a video projector.

This nomadic tool was designed in response to a problem at 1024 Architecture: the need to activate several installations running MadMapper simultaneously, without having to tie up an expensive computer for each project.

The MiniMad targets galleries and museums for long-term exhibitions.

2014

MadMapper 2

 

The new features turn the software into a mapping and VJing tool. MadMapper is now compatible with PS3/PS4 MIDI controllers and joysticks, and it is finally possible to implement 3D objects and surfaces, as well as import SVG lines drawn with other software.

 

Features

  • MIDI protocol
  • 3d objects and surfaces
  • Import of SVG lines
  • LED control in DMX language

2013

MAD Orb

 

An early MadMapper project by 1024 architecture and GarageCube - the unfolded 2D LED matrix is remapped onto the physical 3D object.

A series of specific lighting behaviors are programmed in MadMapper to activate the light architecture in real-time, following the rhythm of the DJs' mixes.
